# Bizarre Industries — companion tools

`BZR / TOOLS / V0.2`

Beyond the editor and the terminal, Bizarre dresses the rest of the bench:

| Tool       | Path                            | What it themes |
|------------|---------------------------------|----------------|
| Zellij     | `terminals/zellij/bizarre.kdl`  | Tab strip, pane frames, modes; lime active-tab indicator |
| Jujutsu    | `tools/jujutsu/config.toml`     | Log/diff colors + `bizarre` log template + brand aliases |
| AeroSpace  | `tools/aerospace/aerospace.toml`| Workspaces, gaps, app rules; lime focus border via JankyBorders |
| ForkLift 4 | `tools/forklift/Bizarre.json`   | File list, sidebars, preview, git decorations, terminal palette |

Each one reserves Signal Lime `#C6FF24` for a single semantic role — the
working-copy marker, the active tab, the focused window, the cursor. Everything
else uses the supporting palette so the lime *means* something when you see it.
