# Bizarre Industries — Sublime Text

`BZR / SUBLIME / V0.1`

```bash
# macOS
cp editors/sublime/*.sublime-color-scheme \
  "$HOME/Library/Application Support/Sublime Text/Packages/User/"

# Linux
cp editors/sublime/*.sublime-color-scheme \
  "$HOME/.config/sublime-text/Packages/User/"
```

Then `Preferences → Color Scheme → Bizarre …`
