# Bizarre Industries — VS Code Themes

`BZR / VSCODE / V0.2`

Five variants. Lime is reserved for function names + diagnostics — the structural hero of code, used scarcely.

## Install (local, dotfiles)

```bash
mkdir -p ~/.vscode/extensions
ln -s "$PWD/editors/vscode" ~/.vscode/extensions/bizarre-industries.bizarre-themes
```

If that extension path already exists, leave it in place or remove it deliberately before creating a replacement link.

Then `Cmd+Shift+P → Color Theme → Bizarre …`

## Variants

| Theme | Use it for |
|---|---|
| Bizarre Void | the default dark register |
| Bizarre Void Hi-Contrast | projector, OLED, accessibility |
| Bizarre Workshop | warmer dark, long sessions |
| Bizarre Paper | the default light register |
| Bizarre Bone | softer light, daylight desk |
